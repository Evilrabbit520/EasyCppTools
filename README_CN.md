# EasyCppTools

[English](README.md)

EasyCppTools顾名思义，就是一个简单的C++工具集，该集合计划包含轻巧Json解析器，轻巧加/解密集合，在线API接口集合。每天持续更新中，希望您参与到其中来(偶尔一两天会摆烂，见谅)。
